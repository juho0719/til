# Why use frontend library(framework)?

- 사실  `React`나 `Vue`가 하는 일을 바닐라 JS로 모두 구현 가능
- 데이터 변화를 화면에 쉽게 적용
- 컴포넌트, 모듈화 하기 쉬움
- 소규모 개발에서는 티가 잘 안나지만, 관리 데이터가 많아지고 대규모 프로젝트로 갈수록 생산성도 좋고 편하게 개발할 수 있음
